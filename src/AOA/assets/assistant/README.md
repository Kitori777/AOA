ALICE assistant assets (runtime)

This folder is the primary location for assistant visuals used by the desktop app.

Required for current UI:
- `alice_white.png` (recommended)

Fallback names supported by code:
- `airi_avatar.png`

Live2D model pack (future full 1:1 mode):
- `model/character.model3.json`
- `model/*.moc3`
- `model/*.motion3.json` (optional but recommended)
- `model/*.physics3.json` (optional)
- `model/*.exp3.json` (optional)
- `model/texture_*.png`
